Task Runner

細々した作業を自動化してしまいましょう。という風潮

Grunt

  • 現在主流のタスクランナー
  • 豊富なプラグイン
  • 多くの日本語情報

Gulp

  • 後発のタスクランナー
  • gruntに比べてサクサク動く（気がする）
  • コンフィグが書きやすい（気がする）
  • ナウい

How to use

必要なファイル

  • package.json
    使いたいパッケージ（Nodeアプリケーション）を記載
  • Gruntfile.js or gulpfile.js
    コンフィグ（タスクの定義等）

実行方法

  1. $npm install
    package.jsonに記載さているパッケージをインストール
  2. $grunt(gulp) example
    grunt(gulp)のみだとdefaltタスクを実行

CSS PreProcesser

言語として貧弱なCSSをいい感じに書きたい。という流れ

CSS Archtect

cssは自由かつ簡単に記述しする事ができる為、肥大化しやすく破綻してしまいやすい。
その為、ルールを作り秩序をもたらす事が必要となる。

代表的な設計思想

OOCSS

Object Oriented Cascading Style Sheets

  • 2009年にNicole Sullivanによって提唱
  • 構造とスキンに分離してクラスを定義する
  • https://fronteers.nl/congres/2009/sessions/object-oriented-css
SMACSS

Scalable and ModularArchitecture for CSS

  • 2011年にJonathan Snookによって提唱
  • Base、Layout、Module、State、Themeの５つのカテゴリに分類しクラスを定義する
  • http://vimeo.com/61755493
BEM

Block、Element、Modifier

  • 2012年にYandex社によって提唱
  • Block、Element、Modifierに分類し、厳格な命名規則に従いクラスを定義する
  • http://vimeo.com/53219242
